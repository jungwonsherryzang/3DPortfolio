# 3DPortfolio
Built my own Portfolio with using 3D objects library

## Enhanced my previous portfolio with adding *3D effects* on it

+Used JavaScript as the main programming language
+Used various libraries and frameworks to make the perfect 3D Portfolio<br>
+ThreeJS: animating applications<br>
+3Drei: turning not 3D into 3D<br>
+TailWindCSS to design: https://tailwindcss.com/docs/guides/vite<br>
+ReactVerticalTimeline library: to elaborate my work experience in vertical timeline<br>
https://www.npmjs.com/package/react-vertical-timeline-component<br>

+Sketchlab: finding the 3D models and used https://gltf.pmnd.rs/ to change 3D model file to jsx<br>
+Worked on cleaned up the unnecessary things and removed meshes and shadows<br>
+EmailJS: to connect with Gmail and "contact" section in the portfolio<br>
+Hostinger: to deploy the portfolio public<br>
